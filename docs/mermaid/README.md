# Mermaid ERs essenciais

Esta pasta contem apenas os diagramas ER essenciais para entendimento estrutural do projeto.

## Diagramas essenciais

- `er-usuarios-auth-pagamentos.mmd`
  - Visao de usuarios, autenticacao e metodos de pagamento.
- `er-carteira.mmd`
  - Visao da carteira (accounts, ledger, intents, withdrawals, receipts).
- `er-markets.mmd`
  - Visao do dominio de mercados (market, categorias, posicoes e transacoes).
- `er-orders-webhooks-auditoria-emails.mmd`
  - Visao de pedidos, webhooks, auditoria e fila de emails.
- `er-portfolio.mmd`
  - Visao do modulo de portfolio (positions, fills, snapshots).

## Escopo

Aqui ficam os ERs essenciais para entendimento do sistema.
